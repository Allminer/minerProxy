# Allminer 
Golang编写性能极高，算法升级（独家），抽水平稳0算力损耗，Web后台实时查看抽力算力，支持本地部署混淆加密客户端，Web后台HTTPS访问。

Web界面操作，简单易用，一键安装，小白可以轻松上手。开机自启动，进程守护运行，自动调整连接数限制。  支持ETH/ETC/BTC/LTC/RVN
